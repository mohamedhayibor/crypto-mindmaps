### Contributing

If you're on OSX:

```sh
$ brew install graphviz
```

Edit the dot file then generate the svg:

```sh
$ dot -Tsvg file.dot > file.svg
```
> if you're on another OS, get the appropriate [download here]( http://www.graphviz.org/Download.php)
